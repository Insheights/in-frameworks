# Insheights 8 Organizational Values Measurement Model — Practitioners Guide

**How to read, diagnose, and act on organizational value generation across all eight dimensions of the IIVCS.**

---

## At a Glance

| | |
|---|---|
| **What it does** | Synthesizes 3E and AER signals into an organizational value profile — revealing what the organization is generating, where value is leaking, and which gaps create compounding drag across other dimensions |
| **When to use it** | At each OSM governance cadence; when value generation patterns need diagnosis; when board or investor reporting requires an IR 6 Capitals translation; when strategic hypotheses need validation against organizational outcomes |
| **Who runs it** | The person accountable for OSM governance — typically a Chief of Staff, Strategy Lead, or Value Creation Lead |
| **Who must be in the room** | OSM governance lead; PTS / delivery lead; CVDM / customer experience lead; finance lead (Financial Value signals); people lead (People Value signals) |
| **Key inputs** | 3E signal maps from instrumented architecture elements; AER data across Customer and Organizational tracks; OSM strategic hypotheses; governance and reporting requirements |
| **Key outputs** | An 8 Values profile; a compounding diagnosis; a prioritized action list; an IR 6 Capitals translation for external reporting |
| **Typical timeframe** | Initial profile: one to two working sessions. Ongoing review: aligned to OSM governance cadence — typically quarterly, with out-of-cycle triggers defined in Step 5 |

> **Not a product organization?** "Product team" → "delivery team"; "customer" → "patient/client/citizen/beneficiary"; "value stream" → "service stream/program stream." The eight value dimensions are universal — only the vocabulary of what generates them shifts by sector.

---

## Before You Start

**Inputs required**

- [ ] 3E signal maps from at least the highest-priority architecture elements — Value Streams and Value Stages at minimum; individual Capabilities and Processes add diagnostic precision
- [ ] AER data for both tracks: Customer AER (Adoption, Engagement, Retention rates) and Organizational AER (Adoption, Engagement, Retention across the workforce and partners)
- [ ] OSM strategic hypotheses — the value priorities and strategic bets the organization is currently executing against
- [ ] Current financial performance data (for Financial Value) and forward-looking pipeline data (CLV, revenue potential)
- [ ] People and workforce indicators — not survey scores alone, but behavioral signals: internal mobility, capability depth, retention rates by role
- [ ] Any existing ESG, mission, or impact reporting (for Impact Value)

**People required**

| Role | Why they must be present |
|---|---|
| OSM governance lead | Owns the strategic hypotheses that the 8 Values profile validates or refutes; owns the return path to strategic reassessment |
| PTS / delivery lead | Source of Operational Value and People Value signals — the product operating model is the primary generator of both |
| CVDM / customer experience lead | Source of Customer Value and Market Value signals — the delivery layer is where customer outcomes are produced or fail to materialize |
| Finance lead | Source of Financial Value signals — current performance and forward-looking pipeline |
| People lead | Source of People Value signals beyond what Org AER captures — capability depth, leadership bench, culture indicators |

**Prepare the room**

- Gather existing signal data in advance — do not enter the session expecting to source signals in the room. The session is for reading the profile and diagnosing compounding relationships, not for finding data
- Know in advance which of the eight values have instrumented signals and which do not — the first action is separating measured values from assumed ones
- Have the OSM strategic hypotheses written down — the 8 Values profile is not useful in isolation; it is most useful in relation to what the organization was trying to achieve
- If this is the first time the organization is running this process, expect Step 1 to reveal that several values have no signals at all. This is the most common and most important finding of the first session

---

## How to Run It

Each step below is an action. Complete them in order — later steps depend on earlier outputs.

---

### Step 1 — Separate measured values from assumed values

**What you do:** For each of the eight values, determine whether the organization has an observable, owned signal — or is asserting the value without evidence. Record the signal type (leading or lagging), the signal source, and the named owner for each dimension.

**Key question this step answers:** *Which of the eight values are we actually measuring, and which are we assuming?*

**Inputs:** All available signal data gathered before the session
**Output:** An 8 Values signal inventory — one row per value, showing: signal exists (yes/no), signal type (leading/lagging/both), signal source, owner

**Guidance:**
- A value is measured only if there is an observable signal with a named owner who reviews it on a defined cadence. A value claimed because an initiative exists ("we run a mentorship program, so we have People Value") is an assumption, not a measurement
- Do not treat survey data as a behavioral signal. Engagement scores and satisfaction surveys are attitudinal — they tell you how people feel, not what they are doing. A People Value signal based only on survey data is a weak signal; it must be paired with behavioral indicators (internal mobility rates, retention by capability area, cross-functional team activation) to be credible
- Values without signals are not zero — they are unknown. The diagnostic distinction matters: a value that is measured and weak requires an intervention; a value that is unmeasured requires instrumentation first
- The most common unmeasured values in most organizations are: Innovation Value (IP and R&D pipeline is tracked financially, but capability expansion and future value stream health are not), Impact Value (ESG reporting exists but is not connected to operational behavior), and Strategic Value (strategy exists but execution effectiveness is not measured distinctly from financial outcomes)

**Signal inventory template:**

| Value | Signal exists | Signal type | Source | Owner |
|---|---|---|---|---|
| Customer Value | | | | |
| Financial Value | | | | |
| People Value | | | | |
| Operational Value | | | | |
| Market Value | | | | |
| Strategic Value | | | | |
| Innovation Value | | | | |
| Impact Value | | | | |

### Step 2 — Map 3E and AER signals to value dimensions

**What you do:** For each signal the organization has — from 3E measurement across architecture elements and from AER across both tracks — translate the signal into the value dimension(s) it most directly surfaces. Build the value signal map.

**Key question this step answers:** *What do our existing 3E and AER signals tell us about each value dimension?*

**Inputs:** 3E signal maps (from all instrumented elements); AER data (Customer and Org tracks); signal inventory (Step 1)
**Output:** A value signal map — showing which signals speak to which values, and in which direction (leading or lagging)

**Guidance:**
- A single signal can speak to more than one value — do not force each signal into one dimension. Customer AER Retention speaks to both Customer Value (behavioral proof of value realized) and Financial Value (revenue sustainability). Map it to both
- 3E signals are primarily leading — they surface before AER and value outcomes. An Efficiency signal from a critical capability is a leading indicator of Operational Value health; it will surface before Organizational AER Adoption begins to decline
- AER signals are the bridge layer — they sit between 3E delivery signals and 8 Values organizational outcomes. Customer AER Adoption is downstream of 3E Efficiency; it is upstream of Customer Value and Financial Value. Read the signal at the right position in the sequence
- The mapping below is the standard translation. Adapt it to the specific elements and signals in your organization, but do not deviate from the directional logic

**Standard signal-to-value translation:**

| Signal Source | Signal | Most Directly Surfaces | Type |
|---|---|---|---|
| **3E Efficiency** | Friction, completion rates, cost-per-activation | Operational Value | Leading |
| **3E Experience** | Customer sentiment, day-2 return, CSAT at step | Customer Value | Leading |
| **3E Effectiveness** | Outcome attainment, goal achievement, retention after outcomes | Strategic Value | Lagging |
| **Customer AER — Adoption** | Trial-to-activation, feature uptake, onboarding completion | Customer Value, Financial Value | Leading |
| **Customer AER — Engagement** | Active usage depth, session frequency, feature breadth | Customer Value, Market Value | Leading |
| **Customer AER — Retention** | 12-month retention, repeat purchase, churn rate | Financial Value, Market Value | Lagging |
| **Customer AER — Recommendation** | Referral rate, organic acquisition, advocacy behavior | Market Value, Financial Value | Lagging |
| **Org AER — Adoption** | New practice uptake, capability activation rate, tool usage | People Value, Operational Value | Leading |
| **Org AER — Engagement** | Cross-functional participation, initiative engagement depth | People Value, Operational Value | Leading |
| **Org AER — Retention** | Retention by capability area, internal mobility, team continuity | People Value, Strategic Value | Lagging |
| **Financial signals** | Revenue, margin, unit economics, pipeline, CLV | Financial Value | Lagging |
| **Market signals** | Brand perception, NPS (market-level), competitive win/loss | Market Value | Lagging |
| **IP and R&D signals** | Pipeline of new value streams, discovery velocity, hypothesis-to-delivery rate | Innovation Value | Leading/Lagging |
| **ESG / mission signals** | Mission advancement indicators, environmental footprint, community contribution | Impact Value | Lagging |

### Step 3 — Build the value profile and read the compounding relationships

**What you do:** Using the value signal map (Step 2), assign a generation status to each value dimension — generating strongly, generating with gaps, generating weakly, or unmeasured. Then read across all eight dimensions together to identify compounding relationships: which weak dimensions are creating drag on other dimensions, and in which direction.

**Key question this step answers:** *What are we generating, what are we not, and which gaps are causing systemic drag?*

**Inputs:** Value signal map (Step 2)
**Output:** An 8 Values profile — current generation status per dimension; a compounding diagnosis — which gaps are upstream causes, which are downstream symptoms

**Guidance:**
- Read the profile as a system, not as eight independent scores. A weak Operational Value and a weak Customer Value appearing simultaneously are almost certainly linked — trace the direction before concluding independently on each
- The four most common compounding patterns to diagnose:

**Pattern 1 — The engine failure:** Org AER Engagement or Retention is declining while Customer Value and Financial Value still appear strong. This is the most dangerous pattern — the human and operational engine is degrading while the visible outputs are holding. Within 6–9 months, Operational Value will erode, and Customer Value delivery will begin to fail. The strong external signals are a lag; the weak internal signals are the early truth.

**Pattern 2 — The delivery misalignment:** Operational Value is strong but Customer Value is weak. The organization has built capability stock but is activating it for internally defined outcomes, not customer outcomes. The gap is in CVDM — how value is delivered at the journey level, not what the organization is capable of.

**Pattern 3 — The formulation-execution gap:** Strategic Value (formulation) appears strong — clear strategy, strong stakeholder alignment — but Operational Value and Customer Value are weak. Strategy is not translating into executable delivery. PTS adoption is the gap: the operating model has not been built to carry the strategy into delivery.

**Pattern 4 — The extraction pattern:** Customer Value and Financial Value are strong, but People Value and Innovation Value are weak. The organization is extracting value from its current customer relationships and workforce faster than it is generating new value. This is sustainable in the short term and dangerous in the medium term — attrition accelerates, innovation pipeline empties, and the next-generation value stream is not being built while the current one is performing.

**Compounding direction reference:**

The 8 Values form a network, not a linear chain. Most values are convergence nodes — receiving from multiple sources and radiating back into the system. People Value is the primary source node; every other value synthesises inputs from multiple sources.

*Bidirectional — solid (direct, continuous):*

| Relationship | What A generates in B | What B generates in A |
|---|---|---|
| Strategic ↔ Innovation | Direction sets where innovation is focused | Disruptive innovation reshapes strategic direction |
| Strategic ↔ Market | Strategic choices shape market positioning | Market signals inform strategic reassessment |
| Strategic ↔ Financial | Strategy determines capital allocation | Financial performance informs strategic capacity |
| Financial ↔ Operational | Financial returns fund capability investment | Operational efficiency generates financial returns |
| Operational ↔ Customer | Capability enables consistent value delivery | Customer signal drives operational improvement |
| People ↔ Impact | Capability and culture generate societal impact | Purpose and impact attract and retain people |
| Customer ↔ Impact | Genuine customer value creates societal benefit | Impact purpose deepens loyalty and advocacy |
| Market ↔ Impact | Market reach amplifies societal impact | Strong impact builds market reputation and trust |

*Bidirectional — dashed (indirect, episodic):*

| Relationship | Note |
|---|---|
| Innovation ↔ Financial | Financial return from innovation routes through operations and customer delivery, not immediately; financial investment in innovation produces returns on a cycle |

*One-way:*

| Relationship | Mechanism |
|---|---|
| People → Operational | People capability is the prerequisite for operational performance |
| People → Innovation | People capability and engagement drive innovation capacity |
| Operational → Strategic | Operational capacity informs strategic feasibility |
| Operational → Impact | How the organisation operates creates or undermines societal impact |
| Customer → Financial | Customer value delivery generates revenue and financial sustainability |
| Customer → Market | Consistent customer outcomes build market reputation |
| Customer → Strategic | Customer outcomes validate or refute strategic hypotheses |

Read the network for root cause: a gap in People Value creates drag on Operational Value and Innovation Value; a weak Operational Value degrades Customer Value delivery and limits Strategic Value execution; Customer Value gaps flow to Financial Value, Market Value, and Strategic Value feedback. The convergence structure means a single upstream gap can create drag in multiple directions simultaneously — trace which convergence node is receiving a degraded input before concluding on cause.

> **Treat these compounding directions as hypotheses, not laws.** The chains above are reasoned diagnostics for *where to look first*, not statistically proven causal links. Use them to form a root-cause hypothesis, then verify it against the actual signals before acting — a traced-upstream cause the evidence does not support is a wrong diagnosis, not a confirmed one. See the Limitations section of the [8 Values model](../8-values.md).

### Step 4 — Prioritize by compounding leverage

**What you do:** Using the compounding diagnosis (Step 3), rank the value gaps identified not by their severity in isolation but by their leverage: how far upstream in the compounding chain does the gap sit, and how many downstream values does it affect?

**Key question this step answers:** *Which gap do we address first to produce the most systemic improvement?*

**Inputs:** 8 Values profile and compounding diagnosis (Step 3)
**Output:** A prioritized action list — value gaps ranked by compounding leverage, with the first action specified for each

**Guidance:**
- The upstream gap is always the higher-priority intervention, even if the downstream symptom is more visible. Fixing Customer Value symptoms while People Value or Operational Value is degrading upstream produces temporary improvement and delayed recurrence
- Use the following prioritization sequence to determine intervention order:

**Priority 1 — Foundation gaps:** Operational Value and People Value. If either is weak, no other value can be consistently delivered. These are the conditions on which everything else depends.

**Priority 2 — Delivery gaps:** Customer Value. If the foundation is sound but Customer Value is weak, the gap is in CVDM — the delivery layer is not connecting capability to customer outcome. This is a journey and value stage problem, not a capability problem.

**Priority 3 — Multiplier gaps:** Market Value and Strategic Value. These amplify what is already being generated. Investing in them before the foundation and delivery layers are sound produces diminishing returns — market position built on inconsistent delivery erodes, and strategy executed through a weak operating model fails.

**Priority 4 — Future gaps:** Innovation Value and Impact Value. These determine what the organization can generate beyond what it delivers today. They are important but not urgent relative to foundation and delivery gaps — an organization that cannot deliver consistently cannot compound innovation or impact reliably.

**Priority 5 — Lagging confirmation:** Financial Value. Financial Value is the most visible and the most lagging. Directly intervening in Financial Value without addressing upstream gaps treats the symptom. The exception: a Financial Value crisis that threatens the organization's survival overrides the prioritization sequence — stabilize financial performance first, then address root causes.

- Each action in the prioritized list must be specific: "Rebuild the onboarding capability to remove IT dependency" is an action. "Improve Operational Value" is not
- Actions must map to a framework owner: Operational Value gaps → PTS; Customer Value gaps → CVDM; Strategic Value gaps → OSM

---

### Step 5 — Feed upstream to governance and outward to reporting

**What you do:** Define the return paths for the 8 Values profile — upstream to OSM governance (internal) and outward to board and investor reporting via IR 6 Capitals (external). Set the review cadence, named owners, and out-of-cycle triggers.

**Key question this step answers:** *How do the eight value dimensions feed governance and reporting, and when does this profile need to be revisited?*

**Inputs:** 8 Values profile, compounding diagnosis, and prioritized action list (Steps 3–4)
**Output:** Governance briefing (internal); IR 6 Capitals translation (external); review cadence with owners and triggers

**Guidance:**
- The internal governance briefing is not a metrics dashboard. It is a reading of which strategic hypotheses the 8 Values profile confirms, which it refutes, and which are still too early to call. The OSM Governance & Measurement layer needs to know whether execution is validating the strategic direction — not just whether individual metrics moved
- The IR 6 Capitals translation uses the bridge defined in the 8 Values model. Apply it as follows:

**IR 6 Capitals translation:**

| Insheights 8 Values | IR 6 Capitals | What to report |
|---|---|---|
| **Customer Value** | Social & Relationship Capital | Customer behavioral signals (AER Adoption, Engagement, Retention); customer satisfaction and trust indicators |
| **Financial Value** | Financial Capital | Current financial performance (revenue, margin, unit economics); forward-looking pipeline (CLV, revenue potential) |
| **People Value** | Human Capital | Workforce capability depth; Org AER Retention by capability area; culture and leadership health |
| **Operational Value** | Manufactured + Intellectual Capital | Capability stock; process maturity; value stream performance; institutional knowledge |
| **Market Value** | Social & Relationship Capital | Brand perception; competitive position; ecosystem relationship strength; advocacy signals |
| **Strategic Value** | Intellectual Capital | Strategy formulation quality; execution effectiveness; portfolio performance; organizational agility |
| **Innovation Value** | Intellectual Capital | IP assets; R&D pipeline; discovery velocity; new value streams in development |
| **Impact Value** | Natural + Social Capital | Mission advancement; ESG performance; environmental footprint; community and societal contribution |

- Set distinct cadences for internal governance and external reporting — they serve different audiences and have different horizons. Internal governance is quarterly at minimum; external IR reporting is annual at minimum but the 8 Values profile that feeds it should be current at each board cycle
- Name a single owner per value dimension for signal ownership. The governance lead owns the profile synthesis; individual value owners are accountable for the signals that feed it

**Return paths and cadences:**

| Return path | What feeds it | Cadence | Owner |
|---|---|---|---|
| **OSM Governance & Measurement** | Full 8 Values profile — confirmed and refuted strategic hypotheses | Quarterly | OSM governance lead |
| **OSM Strategic evolution loop** | Strategic Value and Financial Value effectiveness signals | Quarterly / annually | OSM governance lead |
| **PTS operating model review** | Operational Value and People Value gaps and root causes | Quarterly | PTS / delivery lead |
| **CVDM performance optimization** | Customer Value and Market Value delivery signals | Monthly / quarterly | CVDM / customer experience lead |
| **IR 6 Capitals board reporting** | Full 8 Values profile translated to capital stock | Annually / per board cycle | Finance lead + governance lead |
| **Out-of-cycle trigger — foundation** | Org AER Retention drops or Operational Value signal degrades sharply | Immediate | OSM governance lead |
| **Out-of-cycle trigger — customer** | Customer AER Adoption or Retention drops below threshold | Immediate | CVDM / customer experience lead |
| **Out-of-cycle trigger — financial** | Financial Value current performance drops below operating threshold | Immediate | Finance lead |

## Worked Examples

---

### Example 1 — The Unsustainable Growth Pattern (B2B SaaS)

> **Scenario:** A B2B SaaS company has grown annual recurring revenue by 40% over two years. Customer AER signals are strong. The board is satisfied. But the VP of Engineering is losing senior engineers at an accelerating rate, three of the last five product initiatives have missed their delivery dates, and the product discovery cadence has slowed. The strategy lead applies the 8 Values model to understand whether the organization is generating sustainable value or extracting it.

**Step 1 — Signal inventory**

The team catalogs available signals. Customer AER: strong data, owned by customer success. Financial Value: strong, owned by finance. Org AER Adoption and Engagement: measured at team level through quarterly pulse checks, weak ownership. Org AER Retention: tracked by HR as attrition rate, not connected to capability area. Innovation Value: no signal — IP pipeline is informally tracked by the CTO. Strategic Value: no signal beyond financial performance. Impact Value: ESG statement exists; no operational signals.

Result: Customer Value, Financial Value — measured, strong. People Value — partially measured, signal ownership fragmented. Operational Value — assumed; no direct capability stock signal. Innovation Value — unmeasured. Strategic Value — unmeasured. Market Value — partially measured through NPS. Impact Value — assumed.

**Step 2 — Value signal map**

Customer AER Adoption and Retention → Customer Value (leading, lagging). Financial performance and pipeline → Financial Value (lagging). Engineering attrition rate and missed delivery dates → People Value (lagging), Operational Value (leading). Delivery date miss rate → Operational Value (leading). Product initiative velocity drop → Innovation Value (leading). NPS → Market Value (lagging).

**Step 3 — Value profile and compounding diagnosis**

Profile: Customer Value — generating strongly. Financial Value — generating strongly. People Value — generating with gaps (attrition is a lagging signal; the leading edge has been declining for longer). Operational Value — generating weakly (delivery miss rate and slowing velocity). Innovation Value — unmeasured but declining (informally confirmed by CTO). Strategic Value — cannot assess without distinct signal. Market Value — moderate.

Compounding diagnosis: This is Pattern 4 — the extraction pattern. The organization is generating Customer Value and Financial Value from a workforce and capability base that is degrading. The People Value gap is upstream: senior engineer attrition reduces the capability stock that Operational Value depends on. Reduced Operational Value then reduces delivery quality and velocity, which will produce Customer Value degradation in the next 6–9 months. The strong financial and customer signals are a lag — they reflect the value generated by a capability base that was stronger six months ago. The leading signals point to a different future.

**Step 4 — Prioritized action**

Priority 1 (foundation): People Value. Diagnose the attrition signal by capability area — which roles are leaving, and what is the Org AER Retention pattern for senior engineers specifically? Until People Value is understood at the capability level, no other intervention is well-targeted. Action: instrument Org AER Retention by role and capability area within 30 days; run a structured exit signal analysis with departing engineers.

Priority 2 (foundation): Operational Value. The delivery miss rate is a symptom. Root cause: is the capability to execute the product initiative scope still present, or has attrition degraded it? Action: PTS lead runs a POM capability assessment — mapping delivery team capability stock against the current product portfolio scope.

Priority 3 (future): Innovation Value. The discovery cadence slowdown is the earliest Innovation Value signal. If it is not addressed, the next-generation value stream pipeline empties. Action: PTS lead maps Discover phase capacity and velocity against prior quarters — identify whether the slowdown is resource-driven or prioritization-driven.

Hold: Customer Value and Financial Value interventions. These are strong. Acting on them while the upstream foundation is degrading adds cost without addressing cause.

**Step 5 — Governance and reporting**

Internal governance: The OSM strategic hypothesis under review is revenue growth at current unit economics. The 8 Values profile refutes the hypothesis as currently stated — growth at current rates is not sustainable if People Value and Operational Value are degrading. The governance briefing to the board must reframe: revenue performance is strong; the conditions that generated it are not. Recommended action: reset strategic priority from growth rate to capability investment, for the next two quarters.

IR 6 Capitals: Human Capital — report workforce capability depth and Org AER Retention as a forward-looking risk signal, not just headcount. Intellectual Capital — flag Innovation Value pipeline as underdeveloped; identify as a forward capital risk.

Out-of-cycle trigger: engineering attrition rate above 18% annualized, or delivery miss rate above 40% in any quarter → immediate OSM governance review.

### Example 2 — The Operational Fortress Pattern (Manufacturing)

> **Scenario:** A mid-size manufacturer has invested heavily in process improvement over five years. OEE (Overall Equipment Effectiveness) is above industry benchmark. On-time delivery is at 97%. But market share has declined for three consecutive years, the sales team reports that customers are choosing competitors for "better fit" despite similar pricing, and new product launches are consistently underperforming against plan. The COO applies the 8 Values model to diagnose the gap between operational strength and market outcome.

**Step 1 — Signal inventory**

Operational signals: OEE, on-time delivery, cost-per-unit — strong, owned by operations. Financial Value: revenue declining, margin stable — owned by finance. Customer Value: customer satisfaction surveys run annually, 4.1/5 average — no behavioral signals. Market Value: market share tracked quarterly, declining. Innovation Value: new product launch performance tracked by sales — three consecutive misses. People Value: workforce satisfaction surveys; no capability depth signal. Strategic Value: annual strategic plan exists; no execution effectiveness signal.

Result: Operational Value — measured, strong. Financial Value — measured, deteriorating (revenue). Market Value — partially measured, declining. Customer Value — partially measured (attitudinal only; no behavioral signal). Innovation Value — measured by outcome (launch miss), not by pipeline. People Value — assumed. Strategic Value — unmeasured.

**Step 2 — Value signal map**

OEE and on-time delivery → Operational Value (leading). Revenue decline → Financial Value (lagging). Market share decline → Market Value (lagging). Customer satisfaction surveys → Customer Value (attitudinal — weak signal type). Launch miss rate → Innovation Value (lagging outcome, not leading pipeline). Workforce satisfaction → People Value (attitudinal — weak signal type).

**Step 3 — Value profile and compounding diagnosis**

Profile: Operational Value — generating strongly. Financial Value — generating with gaps (margin stable, revenue declining). Market Value — generating weakly (declining share). Customer Value — unmeasured behaviorally (attitudinal only). Innovation Value — generating weakly. People Value — unmeasured behaviorally. Strategic Value — unmeasured.

Compounding diagnosis: This is Pattern 2 — the delivery misalignment. Strong Operational Value with weak Customer Value and Market Value signals a CVDM gap: the organization is producing efficiently against internally defined specifications, but those specifications are not aligned to what customers value. The customers choosing competitors for "better fit" are telling the story directly — the product is operationally well-made for the wrong outcome definition. Operational Value is generating; it is generating the wrong things.

The Innovation Value gap confirms this: new launches are failing not because the organization cannot build, but because what it is building is not validated against customer outcomes before launch. Discovery is either absent or disconnected from the customer journey.

**Step 4 — Prioritized action**

Priority 1 (delivery gap): Customer Value — but this requires behavioral instrumentation first. The attitudinal survey data is not sufficient to diagnose where the customer journey is failing. Action: instrument Customer AER — specifically, track post-purchase engagement depth and renewal or repeat purchase behavior by product line. Run customer outcome interviews with three to five churned customers to identify where value is failing at the experience level, not just the product specification level.

Priority 2 (delivery gap): CVDM review. Once customer behavioral signals exist, map the customer journey for the two highest-revenue product lines using CVDM. Identify which value stages are failing the Experience and Effectiveness dimensions — this will locate where Operational Value's strength is not translating into Customer Value delivery.

Priority 3 (future gap): Innovation Value. The new product launch miss rate is a downstream outcome of missing customer outcome validation in the discovery phase. Action: PTS lead assesses whether a Discover phase exists and whether it uses customer behavioral signals (not just market research) to validate value propositions before development begins.

Hold: no Market Value investment (brand programs, marketing spend) until Customer Value delivery is understood. Market Value is earned through consistent delivery — investing in brand on top of a customer experience problem accelerates perception damage.

**Step 5 — Governance and reporting**

Internal governance: the OSM strategic hypothesis under review is that operational excellence is the primary competitive differentiator. The 8 Values profile partially refutes it: Operational Value is genuinely strong, but operational excellence is a hygiene factor in this market, not a differentiator. The strategic hypothesis must be updated — differentiation is in the customer outcome layer, not the production layer.

IR 6 Capitals: Manufactured and Intellectual Capital — strong; report operational capability stock as an asset. Social & Relationship Capital — report as a risk: market share decline and customer value gap represent relationship capital at risk, not being built.

Out-of-cycle trigger: market share drops more than two percentage points in any quarter → immediate strategic review. Customer AER Adoption for any new product launch falls below 60% at 90 days → immediate CVDM review of that value stream.

### Example 3 — The Strategy-Execution Disconnect (Professional Services)

> **Scenario:** A professional services firm has completed a major strategy refresh — a clear market positioning, a defined client value proposition, and strong leadership alignment to the new direction. Twelve months later, revenue from the new value proposition is below plan, the delivery teams report they are "still doing the same work as before," and two high-value clients have not renewed. The Managing Partner applies the 8 Values model to diagnose where the strategy is failing to translate into delivery.

**Step 1 — Signal inventory**

Strategic Value: leadership alignment survey completed — 8.2/10. Strategic plan documented and communicated. No execution effectiveness signal. People Value: capability skills assessments run annually; no behavioral engagement signal tied to the new direction. Operational Value: utilization rate tracked (billable hours); no capability-to-strategy alignment signal. Customer Value: client satisfaction surveys, 4.3/5 average. No behavioral AER signal. Financial Value: revenue below plan, pipeline tracked by partner. Market Value: brand perception tracked informally; no structured signal.

Result: Strategic Value — partially measured (formulation quality measured; execution effectiveness unmeasured). People Value — partially measured (capability inventory; no behavioral engagement signal for the new direction). Operational Value — measured for utilization, not for strategic alignment. Customer Value — attitudinal only. Financial Value — measured, underperforming.

**Step 2 — Value signal map**

Leadership alignment → Strategic Value formulation (leading). Revenue below plan → Financial Value (lagging), Strategic Value execution (lagging). Delivery team self-report ("same work as before") → Operational Value (strategic alignment gap — leading). Client non-renewal → Customer Value, Financial Value (lagging). Utilization rate → Operational Value efficiency (neutral — high utilization of the wrong capabilities is not Operational Value generation).

**Step 3 — Value profile and compounding diagnosis**

Profile: Strategic Value — generating with gaps (formulation strong; execution weak). People Value — generating with gaps (capability exists; behavioral engagement with the new direction is absent). Operational Value — generating with gaps (utilization strong; strategic alignment absent). Customer Value — unmeasured behaviorally; client non-renewals are a lagging warning. Financial Value — generating weakly (below plan). Market Value — unmeasured.

Compounding diagnosis: This is Pattern 3 — the formulation-execution gap. Strategic formulation is strong — leadership is aligned, the direction is clear. But PTS has not been applied: the operating model has not been rebuilt to deliver the new value proposition. Delivery teams are using the same capability activation and process patterns as before because nothing has changed at the POM level. The strategy exists at the OSM layer; it has not been operationalized through PTS into the delivery layer. Customer Value cannot change until CVDM has been rebuilt around the new value proposition — which cannot happen until PTS has stood up the capability and team structure to deliver it.

The Financial Value underperformance is the most downstream symptom. The root is in PTS: the product operating model has not been transformed to carry the new strategy into execution.

**Step 4 — Prioritized action**

Priority 1 (foundation): People Value — behavioral engagement. The delivery teams' self-report ("same work as before") is the most important signal. Org AER Adoption of the new delivery model is the leading indicator: are delivery teams using the new capability areas and methods, or defaulting to the old ones? Action: instrument Org AER Adoption — track which delivery teams are actively working within the new value proposition and which are not, by client engagement type.

Priority 2 (foundation): Operational Value — strategic alignment. The PTS lead must assess whether the POM has been updated to reflect the new strategy: do team structures map to the new value streams? Are the capabilities required to deliver the new value proposition present and activated? Action: PTS lead runs a POM alignment assessment within 60 days — current capability and team structure mapped against the new value proposition requirements.

Priority 3 (delivery): CVDM review. Once the POM is aligned, CVDM must be rebuilt around the new client journey — mapping the new value proposition to value stages and ensuring each stage has a defined capability and process activation. The client non-renewals suggest the old journey is failing clients who expected the new value proposition; they must not wait longer than 90 days for visible delivery change.

Hold: Financial Value interventions (sales acceleration, pipeline investment) until the delivery model is rebuilt. Pipeline investment while delivery is misaligned shortens the failure cycle, not the intervention cycle.

**Step 5 — Governance and reporting**

Internal governance: the OSM strategic hypothesis under review is that the new value proposition will generate premium revenue within 12 months. The 8 Values profile refutes the hypothesis on current trajectory — not because the strategy is wrong, but because the operating model has not been built to deliver it. Governance action: reset the 12-month revenue hypothesis; replace it with an execution milestone hypothesis for the next 6 months — POM alignment complete, Org AER Adoption above 70% for new delivery methods, first three new-model client engagements delivered with Customer AER tracked.

IR 6 Capitals: Intellectual Capital — flag the gap between strategic formulation quality and execution effectiveness as a capital risk. Human Capital — flag People Value engagement with the new direction as an investment requirement, not a performance risk.

Out-of-cycle trigger: a second high-value client does not renew → immediate governance escalation. Org AER Adoption of new delivery methods below 50% at 90 days → immediate PTS and OSM review.

## Measurement Connection

### How 3E feeds the 8 Values

3E operates at the architecture element level — Value Streams, Value Stages, Capabilities, Processes, Products/Services. Each 3E signal produced at those levels speaks to specific value dimensions. The 8 Values profile reads across all of those element-level signals and synthesizes them into organizational-level value generation status.

| 3E Dimension | Architecture level | Most directly surfaces |
|---|---|---|
| **Efficiency (leading)** | Capability / Process | Operational Value — friction-free execution is both the measure and the output of operational capability |
| **Efficiency (leading)** | Value Stage / Product/Service | Customer Value — friction at the delivery layer is the first signal that Customer Value is at risk |
| **Experience (leading)** | Value Stage / Product/Service | Customer Value — how value is felt is the most direct Customer Value leading signal |
| **Experience (leading)** | Capability / Process | People Value — how teams experience the operating model reflects People Value health |
| **Effectiveness (lagging)** | Value Stream | Strategic Value — outcome achievement at the Value Stream level validates the strategic hypothesis |
| **Effectiveness (lagging)** | Value Stage / Product/Service | Customer Value — realized customer outcomes confirm or refute value proposition delivery |

### How AER feeds the 8 Values

AER sits between 3E and the 8 Values. It translates 3E operational signals into behavioral confirmation that value is being received and acted on — before that reception registers as organizational value generation.

| AER Signal | Track | Most directly surfaces |
|---|---|---|
| **Adoption** | Customer | Customer Value (leading), Financial Value (leading) |
| **Engagement** | Customer | Customer Value, Market Value |
| **Retention** | Customer | Financial Value (lagging), Market Value (lagging) |
| **Recommendation** | Customer | Market Value (lagging), Financial Value (lagging) |
| **Adoption** | Organizational | People Value (leading), Operational Value (leading) |
| **Engagement** | Organizational | People Value, Operational Value |
| **Retention** | Organizational | People Value (lagging), Strategic Value (lagging) |

### The measurement sequence

```
3E signals (leading — delivery layer)
        ↓
AER signals (bridge — behavioral confirmation)
        ↓
8 Values profile (lagging — organizational value generation)
        ↓
OSM governance (return path — strategic validation)
        ↓
IR 6 Capitals translation (external reporting — capital stock)
```

A gap in any 3E dimension predicts an AER gap before behavioral data confirms it. An AER gap predicts a value dimension gap before the organizational profile reflects it. Reading upstream in the sequence enables earlier intervention — before value generation failures register as organizational outcomes.

## System Health Check

These signals confirm the 8 Values model is being applied correctly — not just being followed procedurally. Review them at each governance cadence.

| Signal | Healthy | At Risk |
|---|---|---|
| **Measured vs. assumed ratio** | At least six of the eight values have an observable, owned signal | More than two values are being claimed without a behavioral or operational signal behind them |
| **Signal ownership** | Each value dimension has a named owner who reviews signals on a defined cadence | Signals are produced and reviewed in a group with no individual accountability for acting |
| **Compounding diagnosis** | When any value dimension is weak, the upstream compounding chain is traced before action is taken | Interventions are taken directly on the symptomatic value without tracing the upstream cause |
| **Prioritization sequencing** | Foundation gaps (Operational Value, People Value) are addressed before multiplier gaps (Market Value) or future gaps (Innovation Value) | Visible or externally reported gaps (Financial Value, Market Value) are prioritized over upstream root causes |
| **Governance return path** | The 8 Values profile is connected to OSM governance with a named cadence and is used to validate or refute specific strategic hypotheses | The profile is produced but not connected to any strategic decision or governance action |

**If two or more signals are At Risk:** Stop reviewing the profile and return to Step 1. The profile being generated is not connected to the value generation system — reassess which values have credible signals, re-establish signal ownership, and reconnect the output to a named governance action before the next review cycle.

## Translation Notes

### Non-product contexts

The eight value dimensions are universal. What shifts by sector is the vocabulary for how each value is generated and what the signals look like in practice.

| IIVCS Default | Healthcare | Government | Professional Services | Manufacturing | Financial Services |
|---|---|---|---|---|---|
| Customer | Patient / client | Citizen / resident | Client / beneficiary | End customer / buyer | Client / policyholder |
| Customer Value | Patient outcome and trust | Citizen service realization and trust | Client outcome and confidence | Product value delivered to buyer | Policy value delivered to client |

> **Core principle:** The eight dimensions — and the compounding relationships between them — are unchanged across sectors. What shifts is the language for what generates each dimension and what its signals look like in context. The diagnostic logic and prioritization sequence apply universally.

## Quick Reference

**Steps**

1. Separate measured values from assumed values — *Which of the eight are we actually measuring?*
2. Map 3E and AER signals to value dimensions — *What do existing signals tell us about each value?*
3. Build the value profile and read compounding relationships — *What are we generating, and which gaps cause systemic drag?*
4. Prioritize by compounding leverage — *Which gap do we address first for the most systemic improvement?*
5. Activate the feedback loop — *How does the profile feed governance and reporting, and when do we revisit?*

**Health signals at a glance**

- At least six of the eight values have an observable, owned signal — not assumed
- Each value dimension has a named owner accountable for acting on signal changes
- When a value is weak, the upstream compounding chain is traced before action is taken
- Foundation gaps (Operational, People) are addressed before multiplier or future gaps
- The 8 Values profile is connected to a named governance action and a defined cadence

**Common compounding patterns**

- **Extraction:** Customer Value and Financial Value strong; People Value and Innovation Value declining — the organization is consuming capability faster than it builds it
- **Delivery misalignment:** Operational Value strong; Customer Value and Market Value weak — capability stock is present but activated for the wrong outcomes
- **Formulation-execution gap:** Strategic Value (formulation) strong; Operational Value and Customer Value weak — strategy exists but the operating model has not been built to deliver it
- **Engine failure:** Org AER Engagement or Retention declining while external signals still appear strong — the human and operational foundation is degrading before the downstream signals confirm it

**Prioritization sequence**

```
Foundation first: Operational Value → People Value
       ↓
Delivery next: Customer Value (CVDM gap)
       ↓
Multipliers: Market Value → Strategic Value
       ↓
Future: Innovation Value → Impact Value
       ↓
Lagging confirmation only: Financial Value
```

**Primary 3E and AER connections**

3E Efficiency → Operational Value | 3E Experience → Customer Value | 3E Effectiveness → Strategic Value

Customer AER → Customer Value, Financial Value, Market Value | Org AER → People Value, Operational Value, Strategic Value

**IR 6 Capitals bridge:** Customer Value + Market Value → Social & Relationship Capital | Financial Value → Financial Capital | People Value → Human Capital | Operational Value → Manufactured + Intellectual Capital | Strategic Value + Innovation Value → Intellectual Capital | Impact Value → Natural + Social Capital
